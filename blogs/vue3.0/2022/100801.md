Vue3 的 setup 中无法使用 this 这个上下文对象，如果非要使用this上的属性和方法，依然可以通过<code>getCurrentInstance</code>方法获取上下文对象：
```js
<script setup>
import { getCurrentInstance } from 'vue'

const { ctx } = getCurrentInstance()
console.log(ctx) // 和 this 的属性一样
</script>
```
