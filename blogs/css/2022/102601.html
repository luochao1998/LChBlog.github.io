冷门又好用的css属性

LCh2022-10-26 15:33:43csscss

# gap

适用于flex和grid布局，设置每个元素之间的间隔

# position: sticky

适用于吸顶需求：标题在滚动的时候，会一直贴着最顶上,场景：比如表格的标题栏、网站的导航栏、手机通讯录的人名首字母标题等等 

// css 部分
.container {
    background-color: oldlace;
    height: 200px;
    width: 140px;
    overflow: auto;
  }
  .container div {
    height: 20px;
    background-color: aqua;
    border: 1px solid;
  }
  .container .header {
    position: sticky;
    top: 0;
    background-color: rgb(187, 153, 153);
}

// html 部分
<div class="container">
  <div class="header">Header</div>
  <div>1</div>
  <div>2</div>
  <div>3</div>
</div>
